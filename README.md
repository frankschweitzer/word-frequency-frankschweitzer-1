# Word Frequency Project

This is a school project to analyze the frequency of words in a given text. The project is implemented in C.

## Table of Contents
- [Installation](#installation)
- [Usage](#usage)
- [Features](#features)
- [Contributors](#contributors)

## Installation

To compile the project, use the following command:

```sh
gcc -o wordfreq wordfreq.c
```

## Usage

Run the compiled program with a text file as an argument:

```sh
./wordfreq <filename>
```

## Features

- Reads a text file and calculates the frequency of each word.
- Outputs the word frequency count to the console.
